# شركة مفتاح النيل للاستثمار والتجارة الدولية (ذ.م.م)
## Nile Key for Investment and International Trade LLC

نظام إدارة متكامل ومؤمن مصمم خصيصاً لتمكين الصادرات الزراعية المصرية في الأسواق العالمية. يجمع التطبيق بين الإدارة التشغيلية اليومية والتحليلات الاستراتيجية الذكية المدعومة بالذكاء الاصطناعي.

### 🚀 المميزات الرئيسية (الإصدار 1.0)

*   **📦 إدارة الشحنات واللوجستيات (إداري فقط):** تتبع شامل للشحنات (بحري، جوي، بري) مع إدارة أرقام الـ ACID ومستندات الشحن بشكل مؤمن.
*   **🚜 قاعدة بيانات الموردين الذكية:** 
    *   الوصول إلى قاعدة بيانات محطات التعبئة العامة في مصر.
    *   التكامل مع **القائمة البيضاء للهيئة القومية لسلامة الغذاء (NFSA)**.
    *   إمكانية إضافة وإدارة موردين خاصين بكل حساب إداري.
*   **🤝 إدارة العملاء الدوليين:** تنظيم بيانات المستوردين، الأكواد الجمركية المفضلة، ونظام ذكي لطلبات عروض الأسعار (RFQ).
*   **🧠 مختبر التحليلات التنبؤية (AI Lab):**
    *   **مستشار الحصاد:** التنبؤ بمواعيد الحصاد المثلى بناءً على المحصول والموقع الجغرافي.
    *   **مكتشف الفرص:** تحليل فجوات العرض والطلب العالمية لتحديد أفضل أسواق التصدير.
    *   **المترجم الدبلوماسي:** صياغة المراسلات التجارية بنبرة احترافية تليق بالتفاوض الدولي.
*   **📚 دليل المصدر المصري:** مركز موارد تعليمي شامل يغطي إجراءات التكويد، التعامل مع منصة "نافذة"، والمعايير الدولية للجودة.
*   **🌐 مركز المواقع الهامة:** نظام ديناميكي لإدارة روابط الهيئات السيادية واللوجستية مع إمكانية التخصيص الكامل للمستخدم.
*   **🔒 نظام صلاحيات متطور (RBAC):** هيكلة احترافية تبدأ من "صاحب الشركة" (صلاحيات مطلقة) وصولاً إلى المدير، الموظف، والعميل.

### 🛠 التقنيات المستخدمة

*   **الإطار البرمجي:** [Next.js 15](https://nextjs.org/) (App Router)
*   **قاعدة البيانات والهوية:** [Firebase](https://firebase.google.com/) (Firestore, Authentication)
*   **محرك الذكاء الاصطناعي:** [Google Genkit](https://github.com/firebase/genkit) (Gemini 2.5 Flash)
*   **التصميم:** [Tailwind CSS](https://tailwindcss.com/) + [ShadCN UI](https://ui.shadcn.com/) (Nile Blue Theme)

### 📁 هيكل المشروع

*   `src/app`: صفحات التطبيق ومساراته (Next.js App Router).
*   `src/components`: مكونات واجهة المستخدم (UI Components).
*   `src/firebase`: إعدادات وخدمات Firebase وتكاملها.
*   `src/ai`: تدفقات العمل الخاصة بالذكاء الاصطناعي (Genkit Flows).
*   `src/context`: مزودي البيانات (Language, Theme, Auth).
*   `docs`: التوثيق وهياكل البيانات.
