# شركة مفتاح النيل للاستثمار والتجارة الدولية (ذ.م.م)
## Nile Key for Investment and International Trade LLC

نظام إدارة متكامل ومؤمن مصمم خصيصاً لتمكين الصادرات الزراعية المصرية في الأسواق العالمية. يجمع التطبيق بين الإدارة التشغيلية اليومية والتحليلات الاستراتيجية الذكية المدعومة بالذكاء الاصطناعي.

### 🚀 المميزات الرئيسية

*   **📦 إدارة الشحنات:** نظام متكامل لتسجيل ومتابعة الشحنات (بحري، جوي، بري) مع تتبع أرقام الـ ACID ومستندات الشحن بشكل دقيق.
*   **🚜 إدارة الموردين والعملاء:** قاعدة بيانات مركزية للمزارع، محطات التعبئة (بما في ذلك التكامل مع القائمة البيضاء لسلامة الغذاء)، والمستوردين الدوليين.
*   **🧠 مختبر التحليلات التنبؤية (AI):**
    *   **مستشار الحصاد:** التنبؤ بمواعيد الحصاد المثلى بناءً على المحصول والموقع الجغرافي.
    *   **مكتشف الفرص:** تحليل فجوات العرض والطلب العالمية لاكتشاف فرص تصديرية جديدة.
*   **📚 دليل المصدر المصري:** مركز موارد شامل وتفاعلي يغطي إجراءات التكويد، التعامل مع منصة "نافذة"، والامتثال للمعايير الدولية.
*   **🔒 نظام صلاحيات متطور:** هيكلة احترافية للمستخدمين (المالك، الإدارة، الموظفين، والعملاء) مع نظام حماية صارم للبيانات الحساسة.
*   **🌐 ثنائي اللغة بالكامل:** واجهة مستخدم متطورة تدعم اللغتين العربية والإنجليزية مع مراعاة كاملة لتجربة المستخدم (RTL/LTR).

### 🛠 التقنيات المستخدمة

*   **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
*   **Backend:** [Firebase](https://firebase.google.com/) (Firestore, Auth, App Hosting)
*   **AI Engine:** [Google Genkit](https://js.genkit.dev/) (Gemini 2.5 Flash)
*   **UI/UX:** Tailwind CSS, ShadCN UI, Lucide Icons
*   **Language Support:** React Context API for i18n

### 🛠 البدء في العمل

1.  **تثبيت التبعيات:**
    ```bash
    npm install
    ```

2.  **تشغيل التطبيق (التطوير):**
    ```bash
    npm run dev
    ```

3.  **تشغيل بيئة Genkit:**
    ```bash
    npm run genkit:dev
    ```
