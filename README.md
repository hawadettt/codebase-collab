# شركة مفتاح النيل للاستثمار والتجارة الدولية (ذ.م.م)
## Nile Key for Investment and International Trade LLC

نظام إدارة متكامل ومؤمن مصمم خصيصاً لتمكين الصادرات الزراعية المصرية في الأسواق العالمية. يجمع التطبيق بين الإدارة التشغيلية اليومية والتحليلات الاستراتيجية الذكية المدعومة بالذكاء الاصطناعي.

### 🚀 المميزات الرئيسية (المسودة الأولى)

*   **📦 إدارة الشحنات واللوجستيات:** تسجيل ومتابعة الشحنات (بحري، جوي، بري) مع تتبع أرقام الـ ACID ومستندات الشحن.
*   **🚜 قاعدة بيانات الموردين الذكية:** 
    *   الوصول إلى قاعدة بيانات محطات التعبئة العامة.
    *   التكامل مع **القائمة البيضاء للهيئة القومية لسلامة الغذاء (NFSA)**.
    *   إمكانية إضافة موردين خاصين.
*   **🤝 إدارة العملاء الدوليين:** تنظيم بيانات المستوردين، الأكواد الجمركية المفضلة، ونظام طلبات عروض الأسعار (RFQ).
*   **🧠 مختبر التحليلات التنبؤية (AI Lab):**
    *   **مستشار الحصاد:** التنبؤ بمواعيد الحصاد المثلى بناءً على المحصول والموقع.
    *   **مكتشف الفرص:** تحليل فجوات العرض والطلب العالمية.
    *   **المترجم الدبلوماسي:** ترجمة المراسلات التجارية بنبرة احترافية.
*   **📚 دليل المصدر المصري:** مركز موارد شامل يغطي إجراءات التكويد، منصة "نافذة"، والمعايير الدولية.
*   **🌐 مركز المواقع الهامة:** نظام ديناميكي لإدارة روابط الهيئات السيادية واللوجستية والتقنية.
*   **🔒 نظام صلاحيات متطور (RBAC):** هيكلة احترافية تبدأ من "صاحب الشركة" (صلاحيات مطلقة) وصولاً إلى الموظفين والعملاء.
*   **🌍 دعم ثنائي اللغة:** واجهة مستخدم كاملة باللغتين العربية والإنجليزية مع دعم اتجاهات القراءة (RTL/LTR).

### 🛠 التقنيات المستخدمة

*   **الخيار البرمجي:** Next.js 15 (App Router)
*   **قاعدة البيانات والهوية:** Firebase (Firestore, Auth)
*   **محرك الذكاء الاصطناعي:** Google Genkit (Gemini 2.5 Flash)
*   **التصميم:** Tailwind CSS + ShadCN UI
